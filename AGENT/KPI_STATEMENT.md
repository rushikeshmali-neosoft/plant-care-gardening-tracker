🌿 Plant Care & Gardening Tracker
==================================

Key Performance Indicators (KPI) Report

PROJECT ACCEPTANCE CRITERIA: All KPIs MUST Passed ✅


1. User Management
-------------------

User Registration 	    Users can create accounts with email and password	 
User Login           	Registered users can log in securely
RBAC	                Role based access of services & APIs, Admin role & User role
Profile Management	    Users can update gardening preferences and experience level	 
Password Reset	        Users can reset forgotten passwords via email	 
Session Management	    User sessions are properly maintained and secured	 

2. Plant Catalog & Inventory
----------------------------

Add Plant	          Create plant entries with species, variety, and purchase details	 
Plant Details	      Store scientific name, common name, and care requirements	 
Location Tracking	  Track where plants are located (indoor, outdoor, room, garden bed)	 
Plant Grouping	      Organize plants into collections (Herb Garden, Succulents, etc.)	 
Plant Status	      Mark plants as active, dormant, gifted, or deceased	 

3. Care Schedule Management
----------------------------

Watering Schedule	    Set and track watering frequency based on plant needs	 
Fertilization Schedule	Plan and log fertilizer applications	 
Pruning/Trimming	    Schedule and track pruning activities	 
Repotting Reminders	    Set reminders for when plants need repotting	 
Seasonal Adjustments	Automatically adjust care schedules by season	 

4. Care Activity Logging
----------------------------

Log Watering	        Record watering events with date, amount, and method	 
Log Fertilization	    Track fertilizer type, amount, and application date	 
Log Pruning	            Record pruning details and techniques used	 
Pest/Disease Tracking	Log pest issues and treatments applied	 
General Notes	        Add observations and notes about plant health	 

5. Growth Tracking
-------------------

Photo Timeline	          Upload and organize photos to track plant growth over time	 
Measurement Logging	      Record height, width, and other growth metrics	 
Bloom Tracking	          Log flowering periods and bloom counts	 
Harvest Tracking	      For edible plants, track harvest dates and yields	 
Growth Charts	          Visualize plant growth metrics with charts	 

6. Reminders & Notifications
----------------------------


Upcoming Tasks	          Display care tasks due today, tomorrow, and this week	 
Custom Reminders	      Set specific reminder times for different plant types	 
Weather Considerations	  Suggest adjusting care based on seasonal weather patterns	 
Email Notifications	      Send reminders for overdue care tasks	 
Mobile Push	              Optional mobile notifications for critical care tasks	 

7. Plant Health Monitoring
----------------------------


Health Indicators	    Track plant health status (Excellent, Good, Poor, Critical)	 
Symptom Logging	        Record signs of stress, disease, or nutrient deficiencies	 
Treatment History	    Track treatments applied and their effectiveness	 
Environmental Factors	Log temperature, humidity, and light conditions	 
Recovery Tracking	    Monitor plant recovery after issues are addressed	 

8. Gardening Knowledge Base
----------------------------

 
Care Guides	         Access plant-specific care instructions and tips	 
Problem Solutions	 Common plant problems and recommended solutions	 
Seasonal Tips	     Gardening advice tailored to current season	 
Plant Database	     Searchable database of common houseplants and garden plants	 
User Tips	         Community-contributed care tips and experiences	 

9. Responsive Design
----------------------

 
Mobile Compatibility	Application works on smartphones (320px+ width)	
Tablet Compatibility	Application works on tablets (768px+ width)	 
Desktop Compatibility	Application works on desktop (1024px+ width)	 
Touch Interactions	    Touch-friendly buttons and controls on mobile	 
Photo Capture	        Easy photo upload directly from mobile camera	 

10. Docker & Deployment
------------------------

Docker Container	        Application runs in a Docker container	 
Docker Compose	            Multi-container setup with database	 
Environment Configuration	Configurable via environment variables	 
Database Persistence	    Data persists across container restarts	 
Production Readiness	    Secure configuration for production deployment	 

11. Testing & Documentation
----------------------------

Unit Tests	         Core business logic has unit test coverage	 
Integration Tests	 API endpoints and database operations tested	 
UI Tests	         Critical user flows have automated UI tests	 
API Documentation	 REST API documented with OpenAPI/Swagger	 
User Guide	         Comprehensive user documentation available	 
Code Comments	     Source code includes meaningful comments
