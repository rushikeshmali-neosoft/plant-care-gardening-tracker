

# 🌿 Plant Care & Gardening Tracker — Core Tech Stack
======================================================
  

🏗️ Architecture
------------------------
- Modular Monolithic Architecture  

- REST API Design



🔧 Backend
------------

- Java 17  

- Spring Boot 3.x  

- Spring Web (REST API)  

- Hibernate + Spring Data JPA  

- Spring Validation   

- Maven



🔐 Security
------------

- Spring Security 

- Username & Password Authentication  

- JWT (Access + Refresh Tokens)  

- BCrypt Password Encoding  



🗄️ Database
------------
- PostgreSQL 16  



⚡ Caching
------------
- EhCache (Hibernate L2 Cache)  



🔌 Real-Time Communication
------------------------
- Spring WebSocket  



🔄 Async Processing
--------------------
- RabbitMQ

- Spring Scheduler  

- Spring Events  



🔍 Search
------------
- PostgreSQL Full-Text Search (FTS)  



📁 File Storage
-----------------

- Spring Multipart (File Upload)  

- Local File Storage 



🎨 Frontend
------------

- Angular 18  

- Angular Material  

- NgRx  

- RxJS  

- Chart.js / ngx-charts  

- Reactive Forms  



📊 Logging & Monitoring
------------------------

- Logback  

- SL4J



🚀 CI/CD & Code Review 
------------------------
- GitHub Actions / GitLab CI  

- SonarQube  



🐳 Deployment
------------
- Docker  

- Docker Compose  
  



🧪 Testing & Documentation
--------------------------

- JUnit  

- Mockito  

- Spring Boot Test  

- OpenAPI (Swagger)
